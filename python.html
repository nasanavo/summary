HEASARC

  • HEASARC Access through python - This notebook goes through a specific science case with a number of examples for accessing HEASARC data, including catalogs, images and spectra, using various VO protocols.   • Using VO Tables from HEASARC - This notebook shows how to list, fetch and look at VO tables from the HEASARC archive and also how to create your own VO Table, which can be used to cross-correlate with other VO Tables. (View or download from GitHub)

  • Simple Cone Search of HEASARC catalogs - Find out what catalogs HEASARC has available for cone searches or search a given table with a given position and radius, getting a VOTable XML result. (View or download from GitHub)

  • Using Images from HEASARC - Search for and retrieve images from the HEASARC's holdings. (View or download from GitHub)

  • Retrieve spectra from HEASARC using Simple Spectral Access protocol - Search for and retrieve spectra from the HEASARC's holdings. (View or download from GitHub)

  • Accessing HEASARC tables through the TAP with ADQL - Expand on using Table Access Protocol (TAP) and the Astronomical Data Query Language (ADQL) that it uses. (View or download from GitHub)
